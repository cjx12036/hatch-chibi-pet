# Contributing

Thanks for helping improve Hatch Chibi Pet.

## Good contributions

- Fixes for frame extraction, transparency, chroma spill, scale drift, or atlas validation.
- New pet-safe Q-style presets that remain readable inside a `192×208` cell.
- Better prompts for identity consistency and the nine Codex animation states.
- Cross-platform setup improvements and clearer installation instructions.

## Pull requests

1. Keep changes focused.
2. Run `quick_validate.py` against `skill/hatch-chibi-pet`.
3. Run Python syntax checks for bundled scripts.
4. For visual changes, include a generated contact sheet, validation summary, and affected preview GIFs.
5. Do not commit private source portraits, API keys, local paths, or generated assets without permission.

## Bug reports

Include the Skill version, Codex surface, operating system, failing state, validation JSON, and a generated contact sheet when available. Redact personal information and avoid uploading the original portrait.
